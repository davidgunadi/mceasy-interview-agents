# QA Engineer - Odoo

## Specific Skill & Qualification
- Pengalaman QA Engineer minimal 1–3 tahun.
- Memahami Functional Testing, Regression Testing, dan UAT.
- Memahami proses testing aplikasi web dan ERP.
- Memiliki pengalaman API Testing menggunakan Postman.
- Mampu membuat Test Scenario, Test Case, dan Bug Report.
- Familiar dengan Git dan Agile/Scrum.
- Pengalaman testing Odoo menjadi nilai tambah.
- Memahami flow bisnis Odoo (CRM, Sales, Purchase, Inventory, Accounting)

## Job Description

- Melakukan pengujian terhadap fitur baru, enhancement, dan bug fix pada sistem Odoo.
- Membuat dan memelihara Test Scenario, Test Case, dan dokumentasi testing.
- Melakukan Functional Testing, Regression Testing, Integration Testing, dan UAT.
- Memastikan fitur yang dikembangkan sesuai dengan requirement bisnis.
- Melakukan validasi data melalui API testing.
- Berkolaborasi dengan Developer, Product Owner, dan Business Analyst dalam proses development.
- Melakukan reproduce issue yang ditemukan user dan memberikan dokumentasi bug yang jelas.
- Memastikan kualitas sistem sebelum deployment ke production.
- Membantu peningkatan proses QA dan standar testing di lingkungan Odoo.

---

## Notes for Talent Acquisition

- Bersedia mengikuti technical test/case study.
- Memiliki pengalaman testing Odoo v15+ (Prefered).
- Pernah menggunakan Jira. (Prefered)
- Memahami SQL/PostgreSQL dasar untuk validasi data. (Prefered)
